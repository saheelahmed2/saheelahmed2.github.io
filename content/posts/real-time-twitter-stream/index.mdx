Stream live tweets across the world in real time on a world map.

This web app attaches to the [Twitter Streaming API ](https://developer.twitter.com/en/docs/tutorials/consuming-streaming-data) and extracts tweets with geo data using an open source python library 
[Tweepy](http://docs.tweepy.org/en/v3.5.0/getting_started.html#introduction) that act as a communicator to Twitter API.

![real time twitter stream](tweet2.png)

## Setup

1. Install [Flask](http://flask.pocoo.org/) a server side framework for Python.

```sh
pip install flask
```

2. Clone the repo 

```sh
https://github.com/saheelahmed2/Real-Time-Twitter-Stream.git   
```

3. Create a Twitter App account  on [Twitter Developer Platform](https://developer.twitter.com/) and use access tokens for api request authentication.

Install [Tweepy](http://docs.tweepy.org/en/v3.5.0/getting_started.html#introduction)

```sh
pip install tweepy
```

3. Go to your `flask` folder open scripts and run `activate` on terminal to start the server.

```sh
cd flask/Scripts
activate
```

4. Now go to repository and run `app.py` on terminal to start the application.

```sh
cd Real-Time-Twitter-Stream
python app.py
```

5. Go to your [http://localhost:8080](http://localhost:8080) And that's it !!
